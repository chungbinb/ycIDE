# 编译器配置说明

ycIDE 使用 C++ 编译器来编译项目。为了方便使用，可以将编译器附带到程序目录中。

## 推荐方案：使用 MinGW-w64

### 下载地址
从以下地址下载 MinGW-w64：
- https://github.com/niXman/mingw-builds-binaries/releases
- 推荐下载: `x86_64-13.2.0-release-posix-seh-ucrt-rt_v11-rev0.7z`

或者从 WinLibs 下载:
- https://winlibs.com/
- 推荐下载: GCC 13.2.0 + LLVM/Clang/LLD/LLDB 17.0.5 + MinGW-w64 11.0.1 (UCRT) - release 3
  - Win64: 选择 "without LLVM/Clang/LLD/LLDB"

### 安装步骤

1. 下载 MinGW-w64 压缩包

2. 解压到 ycIDE 程序目录下，确保目录结构如下：

```
ycIDE/
├── ycIDE.exe
├── compiler/
│   └── mingw64/
│       └── bin/
│           ├── g++.exe
│           ├── gcc.exe
│           ├── ld.exe
│           └── ...其他文件
└── ...其他ycIDE文件
```

或者简化结构：

```
ycIDE/
├── ycIDE.exe
├── mingw64/
│   └── bin/
│       ├── g++.exe
│       └── ...
└── ...
```

3. 重启 ycIDE，编译器将自动被检测到

## 查找顺序

ycIDE 按以下顺序查找编译器：

1. `程序目录\compiler\mingw64\bin\g++.exe`
2. `程序目录\compiler\bin\g++.exe`
3. `程序目录\mingw64\bin\g++.exe`
4. Visual Studio (C:\Program Files\Microsoft Visual Studio\...)
5. 系统已安装的 MinGW (C:\MinGW\bin, C:\mingw64\bin, C:\msys64\...)
6. 系统 PATH 环境变量中的编译器

## 最小安装

如果只需要基本编译功能，可以只保留以下文件：

```
mingw64/
└── bin/
    ├── g++.exe
    ├── gcc.exe
    ├── ld.exe
    ├── as.exe
    ├── ar.exe
    ├── libgcc_s_seh-1.dll
    ├── libstdc++-6.dll
    ├── libwinpthread-1.dll
    └── ...相关dll文件
```

以及必需的库文件：
```
mingw64/
├── bin/
├── lib/
│   └── gcc/
│       └── x86_64-w64-mingw32/
│           └── 13.2.0/  (版本号可能不同)
│               ├── libgcc.a
│               ├── libstdc++.a
│               └── ...
└── x86_64-w64-mingw32/
    └── lib/
        ├── libkernel32.a
        ├── libuser32.a
        └── ...
```

## 验证安装

启动 ycIDE，创建或打开一个项目，点击工具栏上的"运行"按钮。
如果在输出窗口看到"使用内置编译器: ..."，说明编译器已正确配置。
