# MinGW-w64 编译器目录

请将 MinGW-w64 编译器解压到此目录。

## 正确的目录结构

```
compiler/
└── mingw64/
    └── bin/
        ├── g++.exe
        ├── gcc.exe
        ├── ld.exe
        └── ...
```

## 下载地址

推荐从以下地址下载：
- https://github.com/niXman/mingw-builds-binaries/releases
- https://winlibs.com/

下载后解压，将 `mingw64` 文件夹复制到此目录即可。
